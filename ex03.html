Hyperlinks

São referências dentro de um documento em hipertexto para outras partes do mesmo documento ou outros documentos. Os links podem ser:
  • Remotos
  • Internos
  • Locais
  • Endereço de correio eletrônico
  • Arquivos para download

Hyperlinks remotos


São hyperlinks que ligam para outros documentos disponíveis na web. Exemplo: Clique aqui!

Hyperlinks locais


São hyperlinks hospedados em um mesmo servidor. Exemplo: Hyperlink local
Hyperlinks em Imagens
É possível deixar uma imagem clicada direcionando para uma outra referência.


Hyperlinks internos
São hyperlinks que referenciam itens dentro do mesmo documento. Exemplo:

  • Capítulo 1
  • Capítulo 2
  • Capítulo 3
